# 딱맞는지원 — 나에게 맞는 정부지원사업 찾기

내 조건(지역·년차·분야·회사 특징)을 입력하면, 그에 맞는 정부지원사업을 **필터링해 제시**하는 백엔드 없는 클라이언트 웹앱. 데이터는 브라우저 `localStorage`에 저장합니다.

## 주요 기능
- **조건 입력 & 저장** — 지역·업력·분야·회사 특징 (localStorage 보관, 재방문 시 복원)
- **자동 필터·적합도 매칭** — 자격 필터 + 적합도 점수로 맞는 사업만 정렬 제시
- **앱 열 때 갱신 + 신규 표시** — "마지막 확인 이후 새 공고" 배너·`✦ NEW` 배지
- **관심 사업 저장** — 마감 임박 순 관리

## 실행
빌드 과정 없음. `index.html`을 브라우저에서 직접 열면 됩니다.

## 구성
| 파일 | 역할 |
|------|------|
| `index.html` / `style.css` / `script.js` | 앱 본체(SPA) |
| `data.js` | 정부지원사업 데이터 (기업마당 API 스냅샷) |
| `update-data.ps1` | 기업마당 Open API로 `data.js` 자동 갱신 (PowerShell) |
| `README-자동수집.md` | 매일 자동 수집(작업 스케줄러) 설정 안내 |
| `PRD.md` / `PLAN.md` | 제품 요구사항 / 구현 계획 |

## 자동 수집 (선택)
`update-data.ps1`이 기업마당 Open API를 호출해 `data.js`를 갱신합니다.
설정 방법은 [`README-자동수집.md`](README-자동수집.md) 참조. (인증키는 `apikey.txt`에 저장하며 git에 포함되지 않습니다.)
